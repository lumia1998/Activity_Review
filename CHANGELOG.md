# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.3] - 2026-03-10

### 修复
- 修复 Rust 后端 13 个编译警告（剩余 5 个为第三方库警告）
- 修复前端 33 个 A11y 可访问性警告
- 修复模态框缺少键盘事件和 ARIA 角色的问题
- 修复表单 label 未关联控件的问题

### 优化
- Clippy 代码风格优化（format! 字符串、类型参数等）

## [1.0.2] - 2026-02-25

### 优化
- 重构概览页面，优化数据加载和显示逻辑
- 优化应用使用图表组件，提升渲染性能
- 重构隐私设置页面，改善交互体验
- 优化存储设置页面和通用设置页面布局
- 改进关于页面信息展示

### 修复
- 修复活动时长严重偏低的 bug（duration 固定为 5 秒而非实际经过时间）
- 修复"关于"页面数据目录无法正常跳转的问题
- 修复时段分析中的数据统计问题
- 修复部分命令注册和数据库查询问题
- 优化主进程启动流程，提升稳定性

### 其他
- 项目清理：移除残留空目录和构建产物
- 新增应用图标缓存机制
- 以用户视角重写项目文档

## [1.0.1] - 2026-02-24

### 修复
- 修复后端命令注册被覆盖导致部分功能失效的问题
- 修复概览页定时器和事件监听器未清理的内存泄漏
- 修复时段摘要页首次加载重复请求两次数据
- 修复时间线查看历史日期时统计文案错误显示为"今日"
- 修复存储设置页使用原生弹窗，改为应用内 toast 提示

### 优化
- 移除 capabilities 中重复的权限声明

## [1.0.0] - 2026-02-04

### 核心功能
- 智能工作追踪：自动记录应用使用情况
- 活动合并：同一应用连续使用自动合并为单条记录
- 浏览器 URL 追踪：记录访问的网站
- 时间线视图：可视化查看每日工作轨迹
- AI 日报生成：支持本地 Ollama 和云端 API

### 隐私保护
- 应用规则：跳过、模糊或正常记录指定应用
- 敏感关键词过滤：自动过滤包含敏感词的活动
- 域名黑名单：不记录指定网站  
- 锁屏检测：锁屏时自动暂停记录

### 平台支持
- macOS 10.13+：屏幕录制权限检测
- Windows 10+：完整功能支持

### 技术规格
- 后端：Rust + Tauri 2
- 前端：Svelte 4 + TailwindCSS
- 数据库：SQLite
